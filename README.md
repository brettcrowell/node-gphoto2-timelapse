> The best module ever.


## Install

```sh
$ npm install --save solar-lapse
```


## Usage

```js
var solar-lapse = require('solar-lapse');

solar-lapse('Rainbow');
```

```sh
$ npm install --global solar-lapse
$ solar-lapse --help
```

```sh
# creates a browser.js
$ npm run browser
```
